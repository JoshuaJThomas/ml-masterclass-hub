# Question Bank Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce a validated `public/bank/questions.json` of coding exercises covering ALL course chapters (0–23), plus a `meta.json` whose `completedThrough` gates what the app later serves, backed by a schema validator with tests.

**Architecture:** The bank is plain JSON served as a static asset from `public/bank/` (fetchable at `${base}bank/...`). A pure validator function (`scripts/validate-bank.js`) enforces the schema and is unit-tested; a CLI wrapper runs it over the real files and is wired into `npm test` so a malformed bank fails CI. The exercise content itself is generated by reading the course notebooks (see Task 3); the validator is the quality gate.

**Tech Stack:** Node (ESM), Vitest. No new runtime deps.

## File Structure

- `public/bank/meta.json` — `{ completedThrough, currentChapter, generatedAt }`.
- `public/bank/questions.json` — array of exercise objects (the content).
- `scripts/validate-bank.js` — `validateBank(meta, questions)` pure fn + CLI entry.
- `scripts/validate-bank.test.js` — Vitest unit tests for the validator.
- `package.json` — add `validate-bank` script; make `test` also validate the real bank.

## Exercise schema (the contract)

Each entry in `questions.json`:
```json
{
  "id": "ch03-pandas-groupby-01",
  "chapter": 3,
  "topic": "pandas groupby",
  "prompt": "Given DataFrame `df` with columns 'team' and 'points', assign to `result` the average points per team.",
  "starterCode": "import pandas as pd\n# `df` is already defined\nresult = ...",
  "check": "assert list(result.round(1)) == [10.5, 20.0]",
  "hint": "Use groupby on 'team' then an aggregation.",
  "solution": "result = df.groupby('team')['points'].mean()",
  "difficulty": "easy"
}
```
- `id`: unique, non-empty; MUST start with `ch` + zero-padded 2-digit chapter (e.g. `ch03-`).
- `chapter`: integer ≥ 0.
- `topic`, `prompt`, `check`, `solution`: non-empty strings.
- `starterCode`, `hint`: strings (may be empty for `starterCode` only if a from-scratch task; `hint` non-empty preferred).
- `difficulty`: one of `easy` | `medium` | `hard`.
- `check`: a Python snippet (assertions) the runner will execute against the user's `result`/globals in a later plan.

`meta.json`:
```json
{ "completedThrough": 8, "currentChapter": 8, "generatedAt": "2026-06-04" }
```

- [ ] **Step 3: Implement `scripts/validate-bank.js`**:
```js
const DIFFICULTIES = ['easy', 'medium', 'hard'];
const NONEMPTY = ['topic', 'prompt', 'check', 'solution'];

export function validateBank(meta, questions) {
  const errors = [];

  // meta
  if (!Number.isInteger(meta?.completedThrough) || meta.completedThrough < 0) {
    errors.push('meta.completedThrough must be an integer >= 0');
  }
  if (!Number.isInteger(meta?.currentChapter) || meta.currentChapter < 0) {
    errors.push('meta.currentChapter must be an integer >= 0');
  }
  if (typeof meta?.generatedAt !== 'string' || !meta.generatedAt) {
    errors.push('meta.generatedAt must be a non-empty string');
  }

  // questions
  if (!Array.isArray(questions)) {
    errors.push('questions must be an array');
    return { valid: false, errors };
  }

  const seen = new Set();
  for (const q of questions) {
    const where = q?.id ? `[${q.id}]` : '[<no id>]';
    if (typeof q?.id !== 'string' || !q.id) {
      errors.push(`${where} id must be a non-empty string`);
      continue;
    }
    if (seen.has(q.id)) errors.push(`${where} duplicate id`);
    seen.add(q.id);

    if (!Number.isInteger(q.chapter) || q.chapter < 0) {
      errors.push(`${where} chapter must be an integer >= 0`);
    } else {
      const prefix = `ch${String(q.chapter).padStart(2, '0')}-`;
      if (!q.id.startsWith(prefix)) {
        errors.push(`${where} id prefix must be "${prefix}" to match chapter ${q.chapter}`);
      }
    }

    for (const field of NONEMPTY) {
      if (typeof q[field] !== 'string' || !q[field].trim()) {
        errors.push(`${where} ${field} must be a non-empty string`);
      }
    }
    if (typeof q.starterCode !== 'string') errors.push(`${where} starterCode must be a string`);
    if (typeof q.hint !== 'string') errors.push(`${where} hint must be a string`);
    if (!DIFFICULTIES.includes(q.difficulty)) {
      errors.push(`${where} difficulty must be one of ${DIFFICULTIES.join(', ')}`);
    }
  }

  return { valid: errors.length === 0, errors };
}

// CLI: validate the real bank files; exit non-zero on failure.
if (import.meta.url === `file://${process.argv[1]}`) {
  const { readFileSync } = await import('node:fs');
  const meta = JSON.parse(readFileSync('public/bank/meta.json', 'utf8'));
  const questions = JSON.parse(readFileSync('public/bank/questions.json', 'utf8'));
  const { valid, errors } = validateBank(meta, questions);
  if (!valid) {
    console.error(`Bank invalid (${errors.length} errors):`);
    for (const e of errors) console.error('  - ' + e);
    process.exit(1);
  }
  console.log(`Bank valid: ${questions.length} questions across chapters ` +
    `${Math.min(...questions.map((q) => q.chapter))}-${Math.max(...questions.map((q) => q.chapter))}.`);
}
```
